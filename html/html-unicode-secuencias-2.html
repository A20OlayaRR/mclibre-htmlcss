Esta página contiene muestras de los dibujos (banderas, emojis y otros) que se pueden conseguir mediante secuencias ZWJ de caracteres Unidoce.

Antes de leer esta lección se recomienda leer las lecciones:

  • El juego de caracteres UTF-8, que explica qué es un juego de caracteres informático y, concretamente, qué es el juego de caracteres UTF-8
  • Entidades de carácter y entidades numéricas, que explica cómo incluir caracteres y secuencias de caracteres Unicode en una página web

Información contenida en las fichas

Las fichas siguientes muestran los dibujos (emoticonos, emojis y otros) de Unicode. Para cada carácter se muestra:

  • la secuencia de códigos Unicode hexadecimal (el formato es U+HHHH, donde HHHH es un número hexadecimal)
  • el carácter en dos tipos de letra, (si no existe el carácter en la fuente, el navegador muestra el código Unicode en un cuadrado de borde negro)
    • la fuente sans-serif (la fuente aplicada en cada carácter depende del sistema operativo, del navegador e incluso del carácter, por lo que se recomienda ver esta lección en Firefox, Chrome e Internet Explorer)
    • la fuente EmojiOne versión 2.2.7 (septiembre de 2016). Además, la imagen enlaza con la página del proyecto desde la que se puede descargar el icono en formato SVG para utilizarlo como imagen independiente.
  • la entidad numérica de ese carácter (en formato hexadecimal y decimal)
  • el nombre del carácter (en inglés)
