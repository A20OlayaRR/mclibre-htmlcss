background-image y margin en html

En la hoja de estilo de esta página, la etiqueta html tiene definida las propiedades background-image y margin:

html {
  background: url("cuadricula.png");
  margin: 30px;
}

body {
  border: black 2px solid;
  font-family: sans-serif;
  font-weight: bold;
  margin: 0;
}

h1 {
  margin: 0;
}
