Información contenida en las fichas

Las fichas siguientes muestran los dibujos (emoticonos, emojis y otros) de Unicode. Para cada carácter se muestra:

  • el código Unicode hexadecimal (el formato es U+HHHH, donde HHHH es un número hexadecimal)
  • el carácter en dos tipos de letra, (si no existe el carácter en la fuente, el navegador muestra el código Unicode en un cuadrado de borde negro)
    • la fuente sans-serif (la fuente aplicada depende del sistema operativo, del navegador e incluso del carácter)
    • la fuente Twemoji, que utiliza Firefox en Windows 7 (aunque la versión que se muestra en los ejemplos es la proporcionada por Twitter Color Emoji SVGinOT Font, de abril de 2019, por lo que faltan algunos caracteres). Además, la imagen enlaza con la página del repositorio Twemoji desde la que se puede descargar el icono en formato SVG para utilizarlo como imagen independiente.
  • la entidad numérica de ese carácter (en formato hexadecimal y decimal)
  • el nombre del carácter (en inglés)
